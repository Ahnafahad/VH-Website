LexiCore Color System

All design tokens · dark & light values · usage context · CSS code

How tokens work: All colors are CSS custom properties defined in globals.css inside @theme inline { }. Dark mode values are the defaults. Light mode overrides live in .lx-light { } applied to the vocab layout root. Use tokens via Tailwind: bg-[var(--color-lx-surface)] or inline style: style={{ color: 'var(--color-lx-accent-red)' }}.
VH Brand Colors — parent brand, not LexiCore-specific
These are the VH Beyond the Horizon identity colors defined in globals.css → @theme inline. They appear on the main marketing site (headers, CTAs, footers). LexiCore does not use these directly — it uses its own --color-lx-* tokens — but lx-accent-red is a brighter derivative of the same crimson family.

Form Tokens — separate from layout tokens for structural clarity
Use these instead of raw --color-lx-* tokens inside any input, textarea, label, or form container. They are stronger than general card borders, give inputs a clear field-background layer, and route focus state through the brand red — keeping semantic precision between interactive and informational UI.
